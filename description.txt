[h1]more-flags-mod[/h1]
A Stellaris mod, which adds more flags to the game.
This mod is Achievement compatible.

[url=https://github.com/mobergmann/more-flags-mod][img]https://i.imgur.com/D7IfV4Q.jpeg[/img]


[h2]Have a flag suggestion?[/h2]
There are multiple ways how you can suggest new flags:
1. ([b]Prefered[/b]) Fork me on [url=https://github.com/mobergmann/more-flags-mod/issues]GitHub[/url], add the Flag yourself and create a Pull Request
2. ([b]Easiest[/b]) Create an Issue on [url=https://github.com/mobergmann/more-flags-mod/issues]GitHub[/url]
3. Contact me on Steam

[h2]When creating a Flag[/h2]
[h3]File Format[/h3]
The Files needed for this mod are [code single].png[/code] files or [code single].dds[/code] files (make sure the background is transparent).

[h3]Sizes[/h3]
You need to include multiple sizes of your flag:
- 256 x 256 (map)
- 128 x 128 (default)
- 24 x 24 (small)

[h3]Colors[/h3]
The Files need the following colors
- map: white
- default: gold
- small: gold
